Accessibility Test Page

How to test:
  1. Click the Accessibility Checker extension icon
  2. Enter "test-container" as the div class
  3. Make sure all audit options are enabled
  4. Click "Run Analysis"
  5. You should see several accessibility issues highlighted below

4. ARIA Label Test

Button without aria-label - should fail

❌ Input without label - should fail
✅ Input with label - should pass

Expected Results

When you run the analysis on the "test-container" class, you should see:

  • Small text highlighted (11px text)
  • Small buttons highlighted (20x20px buttons and icon buttons)
  • Poor contrast text highlighted (light gray text)
  • Buttons and inputs without proper labels highlighted

Total expected issues: ~7-10 errors
